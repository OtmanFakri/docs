# Project Scope

The HealthStaff project encompasses the development of a comprehensive web-based HR management system tailored for healthcare organizations. This section outlines the key areas and functionalities included in the project scope.

## Key Features

<AccordionGroup>
    <Accordion title="Data Extraction and Reporting">
        Intuitive interface for easy extraction of reports, ensuring optimal accessibility to necessary information for HR personnel.
    </Accordion>
    <Accordion title="Employee Database">
        Maintain a comprehensive database of employee profiles, including personal information and work history, with flexible accessibility based on needs.
    </Accordion>
    <Accordion title="Medical Consultation Tracking">
        Allow HR administrators to easily access records of medical consultations, treatments, and prescriptions to monitor employee health and well-being.
    </Accordion>
    <Accordion title="Workplace Accident Management">
        Provide detailed reporting on workplace accidents, administered treatments, and sick leaves for thorough evaluation and accurate report generation.
    </Accordion>
    <Accordion title="Date and ID Filtering">
        Enable HR staff to easily generate reports for specific time periods and individual employees identified by their unique identifier (Employee ID), allowing for targeted analysis and intervention.
    </Accordion>
    <Accordion title="Role-Based Access Control (RBAC)">
        Implement RBAC to regulate access to HR reports based on user roles and permissions, ensuring data confidentiality and security.
    </Accordion>
</AccordionGroup>

## Technical Scope

<CodeGroup>
    <CodeBlock title="Frontend">
        ```
        - Developed using Angular framework
        - Responsive design for various devices
        - Interactive user interface
        ```
    </CodeBlock>
    <CodeBlock title="Backend">
        ```
        - Developed using FastAPI
        - RESTful API architecture
        - Integration with PostgreSQL database
        ```
    </CodeBlock>
    <CodeBlock title="Deployment">
        ```
        - Containerized using Docker
        - Deployable on local web servers
        - Compatible with modern web browsers
        ```
    </CodeBlock>
</CodeGroup>

<Note>
    While these features are not included in the current scope, the system architecture is designed to allow for future expansions and integrations.
</Note>

By clearly defining the project scope, we ensure that the HealthStaff system focuses on delivering core functionalities that address the critical needs of healthcare HR management while maintaining flexibility for future enhancements.
